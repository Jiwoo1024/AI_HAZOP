# 🛡️ AI 기반 HAZOP 안전 분석 웹 애플리케이션

> 공정 위험성 분석(HAZOP)을 AI로 보조하여 위험요소 식별부터 개선권고사항 도출까지,  
> 안전관리자의 의사결정을 지원하는 스마트 안전 분석 도구입니다.

🔗 **실행 링크**: [ai-hazop-jiwoo.streamlit.app](https://ai-hazop-jiwoo.streamlit.app)

---

## 📌 개발 배경

제11회 위험성평가 경진대회에서 에틸렌 저장탱크를 대상으로 HAZOP 분석을 수행하던 중,  
**Less Level**과 같이 결과가 복잡하게 연쇄되는 편차에서는 개선조치 도출에 30분 이상 소요되었고,  
실효성도 확신하기 어려운 한계에 부딪혔습니다.

이에 글로벌 엔지니어링 컨설팅사 **Vysus Group**의 AI 기반 디지털 HAZOP 솔루션 사례를 참고하여,  
AI를 보조 도구로 활용하는 HAZOP 웹 애플리케이션을 직접 고안·구현하였습니다.

---

## ✨ 주요 기능

### 1. 단일 편차 HAZOP 분석
- 압력·온도·유량·수위 4개 공정 변수의 편차(Deviation) 기반 위험 분석
- 원인(Cause) / 결과(Consequence) / 기존 안전조치(Safeguard) 체계적 분석

### 2. 위험도 평가 (Risk Assessment)
- 발생빈도(Frequency) × 발생강도(Severity) 기반 위험도 자동 산정
- 위험도 매트릭스 기반 등급 판정 및 허용 여부 판단

### 3. AI 기반 개선권고사항 도출
- **KOSHA 가이드**와 **산업안전보건법령**을 AI에 학습시켜 근거 있는 개선조치 제안
- 관련 사고사례 연계 분석으로 실효성 있는 안전조치 도출
- 개선 전·후 위험도 비교를 통한 안전 수준 향상 확인

### 4. 복합 편차 분석
- 2~3개 편차를 동시에 고려하는 복합 위험 시나리오 분석
- 물리적으로 성립 불가능한 편차 조합 사전 필터링으로 오류 방지

## 📊 적용 공정 및 분석 대상

| 구분 | 내용 |
|------|------|
| 대상 공정 | 에틸렌 저장탱크 공정 |
| 분석 Node | Node 1 (저장탱크), Node 2 (이송 펌프) |
| 분석 편차 | 압력·온도·유량·수위 4개 변수 (More/Less/No 등 13개 편차) |
| 참고 법령 | 산업안전보건법, KOSHA Guide |

---

## 🎯 성과

- 복잡한 편차의 개선조치 도출 시간 **30분 → 5분 이내**로 단축
- 제11회 위험성평가 경진대회 **최우수상** 수상 (한국안전학회장상)
